### **Installation Guide**

* Install Node <br>
`npm install
`
* Install React Router <br>
  ` npm install --save react-router-dom
  `
* Install React Bootstrap <br>
  `npm install react-bootstrap bootstrap
  `
* Install Formik <br>
  `npm install formik --save
  `
* Install Emailjs <br>
  `npm install @emailjs/browser --save
  `
